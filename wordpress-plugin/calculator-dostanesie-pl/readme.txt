Calculator of high school recruitments points for primary school students in Poland.

== Description ==

The dostanesie.pl calculator is a tool that allows primary school students calculate their score required in high school recruitment process.
This plugin simplifies way of adding calculator widget to your WordPress site.

**Main features:**
* Calculate recruitment points based on grades, exam results, and achievements
* Responsive design adapting to various devices

Source code is available at [https://github.com/dostanesie-pl/widget](https://github.com/dostanesie-pl/widget)

== Installation ==

# Automated installation:
1. Visit WordPress Admin Dashboard > Plugins > Add New
2. Search for ‘dostanesie.pl calculator’
3. Click “Install now” and then “Activate”

# Install manually:
1. Upload calculator-dostanesie-pl.zip file to the /wp-content/plugins/ directory
2. Go to WordPress admin > Plugins
2. Activate the dostanesie.pl calculator from the Plugins list
