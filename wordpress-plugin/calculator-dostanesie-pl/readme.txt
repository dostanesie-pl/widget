Kalkulator punktów rekrutacyjnych do szkół średnich w Polsce.

== Description ==

Plugin pozwala na umieszczenie kalkulatora punktów rekrutacyjnych do szkół średnich na stronie opartej o WordPress.
Punkty liczone są zgodnie z obowiązującym rozporządzeniem Ministra Edukacji.

Autorami pluginu są twórcy portalu [dostanesie.pl](https://dostanesie.pl).

**Główne funkcje:**

- obliczanie punktów na podstawie uzyskanych ocen, wyników z egzaminu ósmoklasisty oraz osiągnięć dodatkowych takich jak konkursy, olimpiady czy wolontariat,
- responsywny design, który dostosowuje się do różnych urządzeń,
- możliwość porównania wyników z uczniami z poprzednich lat.

== Installation ==

# Automatyczna instalacja:
1. Zaloguj się do panelu administracyjnego WordPressa
2. Przejdź do zakładki Wtyczki > Dodaj nową
3. Wyszukaj ‘dostanesie.pl calculator’
4. Kliknij "Zainstaluj teraz", a następnie "Aktywuj"

# Dodanie widgetu do strony:
1. Utwórz nową stronę lub edytuj istniejącą
2. Wybierz blok "Kalkulator dostanesie.pl"

# Instalacja ręczna:
1. Kliknij "Pobierz"
2. Zaloguj się do panelu administracyjnego WordPressa
3. Przejdź do zakładki Wtyczki > Dodaj nową
4. Kliknij "Wyślij wtyczkę na serwer"
5. Wybierz plik ZIP, który pobrałeś wcześniej
6. Kliknij "Zainstaluj teraz"
7. Po zakończeniu instalacji kliknij "Aktywuj"
